Система учета продаж OctoFood
============

Пример элемента типа "категория":

```
{
  "1": {
    "type": "category",
    "title": "Некоторый текст",
    "picture": "some/picture/path.jpg",
    "items": {}
  }
}
```

Пример элемента типа "блюдо":

```
{
  "1.2": {
    "type": "item",
    "title": "Некоторый текст",
    "picture": "some/picture/path.jpg"
    "cost": 100500
  }
}
```
